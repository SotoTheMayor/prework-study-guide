Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • ul is an unordered list (bullet points), ol is an ordered list (numbers), and li is for each item.
  • When possible, it is important to use semantic HTML elements, because they make your HTML easier to read by other developers and the browser.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A period preceding a selector indicates it is a class selector.
  • Hex # color codes can be found at https://htmlcolorcodes.com/
  • Link a CSS by adding the following to the head element: link rel="stylesheet" href="./assets/style.css"

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all changes in the working branch to the staging area
  • git commit -m "add message to commit": adds a message to the commit
  • git pull origin main: pulls down changes from the base branch
  • git push origin feature/starter-code: creates a new remote branch title "feature/starter-code" or adds changes to an existing branch
